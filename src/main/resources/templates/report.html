Project Report

1. Project Overview

This project involved building an admin dashboard with product and order management functionality. The admin is able to add, modify, hide, and unhide products, and also manage orders.

2. Requirements Met

  • Admin can add new products
  • Admin can modify product details (price, stock, description, image)
  • Admin can hide and unhide products
  • Admin can update order statuses
  • User authentication (Login & Registration)
